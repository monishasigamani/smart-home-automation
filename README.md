# Smart Home Automation System

An ESP32-based smart home automation system that controls appliances automatically using PIR motion detection.

## 🛠️ Tech Stack
- **Microcontroller:** ESP32
- **Firmware:** Embedded C (Arduino framework)
- **Sensor:** PIR Motion Sensor
- **Platform:** Wokwi Simulator
- **Concepts:** GPIO Interrupts, Actuator Control

## ✨ Features
- Interrupt-driven PIR motion detection
- Automated ON/OFF control of appliances
- Sensor-to-actuator response time: < 100ms
- Modular firmware with reusable driver abstractions
